# fiftyone-visual-agents-lab

A clean and professional experiment repository for reproducing and extending visual agent workflows inspired by the GitHub repository: :contentReference[oaicite:0]{index=0} `harpreetsahota204/odsc_workshop_visual_agents`.

This lab is intentionally kept minimal in the beginning so experiments can grow naturally as progress is made.

---

## Inspiration

This repository is inspired by:

- `https://github.com/harpreetsahota204/odsc_workshop_visual_agents`

The goal is to use the same experimentation style demonstrated there for hands-on learning, structured testing, and future expansion.

## Purpose

This repository is currently focused on:

1. Learning the visual agent workflow step by step
2. Reproducing the workshop experiments in a clean personal lab
3. Running GUI and visual similarity experiments with FiftyOne
4. Expanding notebook-by-notebook as new ideas are tested

---

## Current Notebook

### 1. `Visual Similarity Search Demo with FiftyOne + CLIP.ipynb`

This first notebook will focus on:

* loading image samples
* extracting CLIP embeddings
* running visual similarity search
* exploring nearest neighbors
* using FiftyOne for interactive visualization
* understanding how visual agents retrieve related UI or image elements

This acts as the foundation notebook for all future experiments.

---

## Suggested Next Growth Path

As the lab grows, future notebooks can naturally extend into:

* dataset curation
* GUI automation experiments
* visual grounding
* agent planning
* action prediction
* evaluation and failure analysis

For now, the repository stays intentionally lightweight.
